# Ставропольский политехнический колледж - веб-сайт

## О проекте

Статический веб-сайт для Ставропольского политехнического колледжа. Разработан в рамках курсового проекта по дисциплине «Информационные технологии и программирование».

Сайт содержит следующие разделы:

- Главная страница с приветственным баннером и статистикой
- Новости колледжа
- Специальности с возможностью раскрытия подробного описания
- Информация о колледже и достижениях
- Расписание занятий с переключением групп
- Форма обратной связи с валидацией и сохранением в localStorage

## Технологии

- **HTML5** - семантическая разметка страниц
- **CSS3** - адаптивный дизайн, CSS Grid, Flexbox, медиавыражения
- **JavaScript (Vanilla JS)** - интерактивные элементы, работа с localStorage, табы, мобильное меню

## Структура проекта

**college-website/**
- **index.html** - Главный HTML-файл
- **README.md** - Документация проекта
- **css/**
  - style.css - Все стили сайта
- **js/**
  - script.js - JavaScript логика сайта

## Функциональность

### Реализованные сценарии (согласно диаграммам деятельности):

1. **Просмотр расписания** - переключение вкладок с группами, сохранение выбранной группы в localStorage
2. **Поиск информации о специальности** - карточки специальностей с кнопкой «Подробнее» (JavaScript раскрывает полное описание)
3. **Отправка сообщения через форму обратной связи** - валидация полей (имя, email, сообщение), сохранение в localStorage

### Дополнительно:

- Адаптивный дизайн (мобильные устройства, планшеты, десктопы)
- Гамбургер-меню на мобильных устройствах
- Анимация появления элементов при скролле (reveal-эффект)
- Счётчики статистики с анимацией

## Запуск проекта

### Локальный запуск:

1. Скачайте или склонируйте репозиторий
2. Откройте файл `index.html` в любом современном браузере (Google Chrome, Mozilla Firefox, Safari, Edge)
